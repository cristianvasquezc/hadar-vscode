# Change Log

All notable changes to the "hadar-theme-pack" extension pack will be documented in this file.

## [1.3.2]

- **Migración a Extension Pack**:
  - Esta extensión ha evolucionado para convertirse en un paquete ("Pack").
  - Se ha eliminado el código fuente del tema y los iconos de este repositorio para modularizar el proyecto.
  - Ahora actúa como un instalador que agrupa las nuevas extensiones independientes:
    - `cristianvasquez.hadar-theme` (Tema de color)
    - `cristianvasquez.hadar-icons` (Iconos)
- **Limpieza**: Eliminación de archivos antiguos y optimización del peso de la extensión.
